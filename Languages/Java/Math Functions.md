### 1. **Basic Arithmetic Functions**

- `Math.abs(x)` — Returns the absolute value of `x`.
    
- `Math.max(x, y)` — Returns the greater of `x` and `y`.
    
- `Math.min(x, y)` — Returns the smaller of `x` and `y`.
    
- `Math.addExact(x, y)` — Returns the sum of `x` and `y`, throwing an exception if overflow occurs.
    

### 2. **Rounding Functions**

- `Math.round(x)` — Rounds the floating-point value `x` to the nearest long.
    
- `Math.ceil(x)` — Returns the smallest integer greater than or equal to `x`.
    
- `Math.floor(x)` — Returns the largest integer less than or equal to `x`.
    
- `Math.rint(x)` — Returns the integer closest to `x` as a `double`, rounding halfway cases away from zero.
    

### 3. **Exponentiation and Logarithms**

- `Math.pow(x, y)` — Returns `x` raised to the power of `y`.
    
- `Math.exp(x)` — Returns the value of `e` raised to the power of `x`.
    
- `Math.log(x)` — Returns the natural logarithm (base `e`) of `x`.
    
- `Math.log10(x)` — Returns the base 10 logarithm of `x`.
    

### 4. **Trigonometric Functions**

- `Math.sin(x)` — Returns the sine of the angle `x` (in radians).
    
- `Math.cos(x)` — Returns the cosine of the angle `x` (in radians).
    
- `Math.tan(x)` — Returns the tangent of the angle `x` (in radians).
    
- `Math.asin(x)` — Returns the arc sine of `x`, in radians.
    
- `Math.acos(x)` — Returns the arc cosine of `x`, in radians.
    
- `Math.atan(x)` — Returns the arc tangent of `x`, in radians.
    

### 5. **Random Numbers**

- `Math.random()` — Returns a random number between `0.0` (inclusive) and `1.0` (exclusive).
    

### 6. **Constants**

- `Math.PI` — The value of `π` (approximately `3.14159`).
    
- `Math.E` — The value of `e` (approximately `2.71828`).
    

### 7. **Other Functions**

- `Math.sqrt(x)` — Returns the square root of `x`.
    
- `Math.cbrt(x)` — Returns the cube root of `x`.
    
- `Math.hypot(x, y)` — Returns the Euclidean distance `sqrt(x^2 + y^2)`.
