* HTML (Hypertext Markup Language) is the standard language used to create and structure content on the web. It is a markup language, which means it uses tags to define elements and structure the content of a web page.
* HTML tags are not case sensitive.
* **Absolute URL** - Links to an external image that is hosted on another website.
		Example: src="https://www.w3schools.com/images/img_girl.jpg".
* **Relative URL** - Links to an image that is hosted within the website. Here, the URL does not include the domain name. If the URL begins without a slash, it will be relative to the current page.
		Example: src="img_girl.jpg"
	If the URL begins with a slash, it will be relative to the domain.
		Example: src="/images/img_girl.jpg".


Mostly Used Tags:
* **!DOCTYPE html** - defines that this document is an HTML5 document. helps browsers to display web pages correctly.
* **html** - root element of an HTML page.
* **title** - title for the HTML page (which is shown in the browser's title bar or in the page's tab)
* img src="Link of image/image name in local path" alt="alternative text" width=" " height=" "
* a href="link" - to redirect to a link(hyperlink). href(**Hypertext Reference**.)
* **br** - line break, and is an empty element without a closing tag.
* **style** - add styles to an element, such as color, font, size, and more. syntax= <_tagname_ style="_property_:_value;_">
* **html lang="en"** - tag **does not automatically translate the content** of your website. It simply indicates that the content is in the Telugu language, which can be helpful for search engines, screen readers, and other tools that need to know the language of the text.
* **hr** - displays as a horizontal rule line.
* **pre** - displayed in a fixed-width font. it preserves both spaces and line breaks.
