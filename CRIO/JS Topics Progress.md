### **JavaScript Learning Checklist**

#### Level 1: Foundations (Beginner)

**Basics of JavaScript**
- [ ] Variables (`var`, `let`, `const`)
- [ ] Data Types (Number, String, Boolean, Null, Undefined, Symbol, BigInt)
- [ ] Type conversion (implicit vs explicit)
- [ ] Operators (arithmetic, comparison, logical)

**Control Flow**
- [ ] Conditional statements (`if`, `else if`, `switch`)
- [ ] Loops (`for`, `while`, `do...while`, `for...of`, `for...in`)

**Functions**
- [ ] Functions (declaration, expression, arrow functions, default params)
- [ ] Closures, Variable Shadowing

#### Level 2: Working with Data (Core)

**Strings**
- [ ] Strings & Template Literals
- [ ] String methods (`slice`, `substring`, `replace`, `split`, etc.)

**Arrays**
- [ ] Arrays basics (indexing, iterating, multidimensional arrays)
- [ ] Array basic methods (`push`, `pop`, `shift`, `unshift`)
- [ ] Array advanced methods (`map`, `filter`, `reduce`, `find`, `some`, `every`, `sort`)

**Objects**
- [ ] Objects (create, properties, methods, nested objects)
- [ ] Array of objects
- [ ] Objects vs Arrays

**Operators & ES6 Features**
- [ ] Spread & Rest operators
- [ ] De-structuring (arrays & objects)


#### Level 3: Deeper Understanding (Intermediate)

**Scope & Execution**
- [ ] Scope (block, function, global)
- [ ] Hoisting
- [ ] Temporal Dead Zone (TDZ)

**Special Concepts**
- [ ] Closures
- [ ] Pass by value vs Pass by reference
- [ ] Shallow copy vs Deep copy

**Math & Numbers**
- [ ] Numbers: floating point quirks, Special numbers - `NaN`, `Infinity`
- [ ] Math functions (`Math.random`, `Math.floor`, etc.)

#### **Level 4: Asynchronous & Advanced JS**

**Callbacks & Asynchronous JS**
- [ ] Event loop & call stack
- [ ] Callbacks
- [ ] Promises (`then`, `catch`)
- [ ] Async/Await

**Error Handling**
- [ ] Error handling (`try...catch`, custom errors)

**Advanced Array Usage**
- [ ] Higher-order functions & array callbacks
- [ ] Functional programming patterns

**This & Prototypes**
- [ ] The `this` keyword (binding rules)
- [ ] Prototypes & prototypal inheritance
- [ ] Classes (`class`, `extends`, `super`)

#### Level 5: Browser & Environment

**DOM Manipulation**
- [ ] DOM manipulation (select, update, remove elements)
- [ ] Event listeners & event delegation

**Browser APIs**
- [ ] Fetch API (GET/POST requests, JSON handling)
- [ ] LocalStorage & SessionStorage
- [ ] Cookies basics

**RegExp**
- [ ] Regular Expressions (RegExp)
- [ ] Common use cases (validation, searching, replacing)

**Performance**
- [ ] Performance optimization (debouncing, throttling)
- [ ] Memory management basics

#### Level 6: Becoming Pro

**Modular JavaScript**
- [ ] ES6 Modules (`import`, `export`)
- [ ] CommonJS (Node.js basics)

**Design Patterns**
- [ ] Module, Factory, Singleton, Observer

**Testing**
- [ ] Unit testing (Jest, Mocha, etc.)
- [ ] Debugging skills (DevTools, breakpoints)

**Modern JS & Beyond**
- [ ] Modern ES features (optional chaining, nullish coalescing, async generators)
- [ ] TypeScript basics
- [ ] Tooling (Babel, Webpack, ESLint, Prettier)

**Real-World Pro Skills**
- [ ] Working with APIs (REST, GraphQL)
- [ ] Security basics (XSS, CSRF)
- [ ] Service workers / Progressive Web Apps
- [ ] WebSockets (real-time apps)
