#### Common JavaScript String Methods

| Method                     | Description                                                                                    | Returns                                                  | Accepts a Function?                        |
| -------------------------- | ---------------------------------------------------------------------------------------------- | -------------------------------------------------------- | ------------------------------------------ |
| **Accessing & Extracting** |                                                                                                |                                                          |                                            |
| `at(index)`                | Gets the character at a given index. Accepts negative indices to count from the end.           | A new **string** containing the single character.        | ❌ No                                       |
| `charAt(index)`            | Gets the character at a given index. Does not accept negative indices.                         | A new **string** containing the single character.        | ❌ No                                       |
| `charCodeAt(index)`        | Gets the 16-bit Unicode value (0-65535) of the character at the index.                         | A **number** representing the UTF-16 code unit.          | ❌ No                                       |
| `slice(start, end)`        | Extracts a section of the string and returns it as a new string.                               | A new **string**.                                        | ❌ No                                       |
| `substring(start, end)`    | Similar to `slice()`, but cannot accept negative indices.                                      | A new **string**.                                        | ❌ No                                       |
| `split(separator)`         | Splits a string into an array of substrings based on a specified separator.                    | An **array** of strings.                                 | ❌ No                                       |
| **Searching & Checking**   |                                                                                                |                                                          |                                            |
| `includes(searchStr)`      | Determines whether one string may be found within another string.                              | A **boolean** (`true` or `false`).                       | ❌ No                                       |
| `startsWith(searchStr)`    | Checks if a string begins with the characters of a specified string.                           | A **boolean** (`true` or `false`).                       | ❌ No                                       |
| `endsWith(searchStr)`      | Checks if a string ends with the characters of a specified string.                             | A **boolean** (`true` or `false`).                       | ❌ No                                       |
| `indexOf(searchStr)`       | Finds the index of the **first** occurrence of a specified value.                              | A **number** (the index), or `-1` if not found.          | ❌ No                                       |
| `lastIndexOf(searchStr)`   | Finds the index of the **last** occurrence of a specified value.                               | A **number** (the index), or `-1` if not found.          | ❌ No                                       |
| `match(regexp)`            | Retrieves the result of matching a string against a regular expression.                        | An **array** of matches, or `null` if no match is found. | ❌ No                                       |
| `search(regexp)`           | Executes a search for a match between a regular expression and this string.                    | A **number** (the index of the match), or `-1`.          | ❌ No                                       |
| **Modifying & Creating**   |                                                                                                |                                                          |                                            |
| `concat(str1, ...)`        | Combines the text of two or more strings and returns a new string.                             | A new **string**.                                        | ❌ No                                       |
| `repeat(count)`            | Creates a new string containing a specified number of copies of the original string.           | A new **string**.                                        | ❌ No                                       |
| `replace(pattern, repl)`   | Replaces the **first** match of a pattern (string or regex) with a replacement.                | A new **string**.                                        | ✅ **Yes**, the replacer can be a function. |
| `replaceAll(patt, repl)`   | Replaces **all** matches of a pattern with a replacement.                                      | A new **string**.                                        | ✅ **Yes**, the replacer can be a function. |
| `toLowerCase()`            | Converts the entire string to lowercase letters.                                               | A new **string**.                                        | ❌ No                                       |
| `toUpperCase()`            | Converts the entire string to uppercase letters.                                               | A new **string**.                                        | ❌ No                                       |
| `trim()`                   | Removes whitespace from both ends of a string.                                                 | A new **string**.                                        | ❌ No                                       |
| `trimStart()`              | Removes whitespace from the beginning of a string.                                             | A new **string**.                                        | ❌ No                                       |
| `trimEnd()`                | Removes whitespace from the end of a string.                                                   | A new **string**.                                        | ❌ No                                       |
| `padStart(len, padStr)`    | Pads the current string from the start with another string until it reaches the target length. | A new **string**.                                        | ❌ No                                       |
| `padEnd(len, padStr)`      | Pads the current string from the end with another string until it reaches the target length.   | A new **string**.                                        | ❌ No                                       |
