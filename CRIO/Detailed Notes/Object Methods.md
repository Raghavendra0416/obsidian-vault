
JavaScript Object Methods — With Returns & Examples

| Category                            | Method(s)                                    | Description                               | Returns                         | Example                                                                                                                                   |
| ----------------------------------- | -------------------------------------------- | ----------------------------------------- | ------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| **Property Enumeration**            | `Object.keys(obj)`                           | Get all keys of an object                 | **Array of strings**            | `js\nconst user = {id: 1, name: "Raghav"};\nconsole.log(Object.keys(user)); // ["id", "name"]`                                            |
|                                     | `Object.values(obj)`                         | Get all values of an object               | **Array of values**             | `js\nconsole.log(Object.values(user)); // [1, "Raghav"]`                                                                                  |
|                                     | `Object.entries(obj)`                        | Get key–value pairs                       | **Array of [key, value] pairs** | `js\nconsole.log(Object.entries(user)); // [["id", 1], ["name", "Raghav"]]`                                                               |
| **Object Creation & Cloning**       | `Object.assign(target, ...sources)`          | Copy properties into target               | **Modified target object**      | `js\nconst a = {x:1}, b = {y:2};\nconsole.log(Object.assign(a, b)); // {x:1, y:2}`                                                        |
|                                     | `Object.create(proto)`                       | Create new object with given prototype    | **New object**                  | `js\nconst proto = {greet() {return "hi"}};\nconst obj = Object.create(proto);\nconsole.log(obj.greet()); // "hi"`                        |
| **Immutability & Property Control** | `Object.freeze(obj)`                         | Prevent all changes                       | **Same frozen object**          | `js\nconst car = {brand:"Tata"};\nObject.freeze(car);\ncar.brand="BMW";\nconsole.log(car.brand); // "Tata"`                               |
|                                     | `Object.seal(obj)`                           | Prevent adding/removing, but allow modify | **Same sealed object**          | `js\nconst phone={model:"iPhone"};\nObject.seal(phone);\nphone.model="Pixel";\nconsole.log(phone.model); // "Pixel"`                      |
|                                     | `Object.preventExtensions(obj)`              | Prevent adding new props                  | **Same object**                 | `js\nconst book={title:"JS"};\nObject.preventExtensions(book);\nbook.author="Me";\nconsole.log(book.author); // undefined`                |
| **Property Definition & Metadata**  | `Object.defineProperty(obj, prop, desc)`     | Define property with descriptor           | **Same object**                 | `js\nconst obj={};\nObject.defineProperty(obj,"age",{value:25});\nconsole.log(obj.age); // 25`                                            |
|                                     | `Object.getOwnPropertyDescriptor(obj, prop)` | Get property descriptor                   | **Descriptor object**           | `js\nconsole.log(Object.getOwnPropertyDescriptor(obj,"age"));\n// { value: 25, writable: false, enumerable: false, configurable: false }` |
|                                     | `Object.getOwnPropertyNames(obj)`            | Get all property names                    | **Array of strings**            | `js\nconsole.log(Object.getOwnPropertyNames(obj)); // ["age"]`                                                                            |
|                                     | `Object.getOwnPropertySymbols(obj)`          | Get all symbol properties                 | **Array of symbols**            | `js\nconst sym=Symbol("id");\nconst obj2={[sym]:123};\nconsole.log(Object.getOwnPropertySymbols(obj2)); // [Symbol(id)]`                  |
|                                     | `Object.getOwnPropertyDescriptors(obj)`      | Get all descriptors                       | **Object of descriptors**       | `js\nconsole.log(Object.getOwnPropertyDescriptors(obj));\n// { age: { value:25, writable:false, ... } }`                                  |
| **Prototype Handling**              | `Object.getPrototypeOf(obj)`                 | Get prototype of object                   | **Prototype object**            | `js\nconsole.log(Object.getPrototypeOf([]) === Array.prototype); // true`                                                                 |
|                                     | `Object.setPrototypeOf(obj, proto)`          | Set prototype of object                   | **Modified object**             | `js\nconst animal={speak(){return "sound"}};\nconst dog={};\nObject.setPrototypeOf(dog, animal);\nconsole.log(dog.speak()); // "sound"`   |
| **Equality Checks**                 | `Object.is(val1, val2)`                      | Compare values strictly                   | **Boolean**                     | `js\nconsole.log(Object.is(NaN, NaN)); // true\nconsole.log(Object.is(+0, -0)); // false`                                                 |
| **Grouping (ES2024)**               | `Object.groupBy(iterable, callback)`         | Group elements by callback result         | **Object**                      | `js\nconst nums=[6.1, 4.2, 6.3];\nconsole.log(Object.groupBy(nums, Math.floor));\n// { 4: [4.2], 6: [6.1, 6.3] }`                         |
| **Own-Property Check**              | `Object.hasOwn(obj, prop)`                   | Check if property exists directly         | **Boolean**                     | `js\nconst car={brand:"Tata"};\nconsole.log(Object.hasOwn(car, "brand")); // true\nconsole.log(Object.hasOwn(car, "toString")); // false` |
