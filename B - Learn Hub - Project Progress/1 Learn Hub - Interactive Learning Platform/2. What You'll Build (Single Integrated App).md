
Think of it like a mini **Udemy + LeetCode + Khan Academy** combined:
### **Main Pages/Sections:**

1. **Landing Page** (Home)    
    - Hero section with search functionality
    - Featured courses carousel
    - Statistics dashboard (total users, courses, etc.)
2. **Course Catalog**
    - Filter/sort courses (by difficulty, category, rating)
    - Search with debouncing
    - Card-based layout
3. **Course Detail Page**
    - Video player mockup
    - Course curriculum (expandable sections)
    - Progress bar
    - Review/rating system
4. **Code Playground**
    - Interactive code editor
    - Algorithm visualizer
    - Run test cases
    - Submit solutions
5. **Quiz Section**
    - Multiple choice questions
    - Timer functionality
    - Score calculation
    - Results with explanations
6. **User Dashboard**
    - Progress tracking (charts/graphs)
    - Certificates earned
    - Learning streak calendar
    - Bookmarked courses
7. **Profile Page**
    - User settings
    - Achievement badges
    - Activity history
