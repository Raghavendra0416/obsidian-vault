A web app that combines:
1. **Algorithm Visualizer** - Visualize sorting, searching, recursion
2. **Code Practice Section** - Solve problems with hints/solutions
3. **Performance Dashboard** - Track progress with charts
4. **Study Resources** - Notes on DS/Algo concepts

## ðŸ’¡ Key Features & Concepts Used

### 1. **Algorithm Visualizer**

- **Concepts**: Arrays, loops, recursion, closures, async/await
- **Algorithms**:
    - Sorting (bubble, selection, merge, quick)
    - Binary Search with bounds
    - Two Pointer technique
    - Sliding Window

### 2. **Code Practice Section**

- **Concepts**: Objects, array of objects, filter, map, reduce
- **Features**:
    - Problem list with difficulty levels
    - Code editor (textarea with syntax highlighting)
    - Test cases validation
    - Hints system using closures

### 3. **Performance Dashboard**

- **Concepts**: Data visualization, array methods, destructuring
- **Metrics**:
    - Problems solved (by difficulty)
    - Time complexity analysis
    - Progress charts

### 4. **Data Management**

- **Concepts**: LocalStorage, JSON, deep/shallow copy
- Save user progress, solved problems, notes
